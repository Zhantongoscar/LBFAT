## 二、全部可能名称速查表（按功能分类）

### 2.1 地（GND）
| 名称 | 备注 |
|------|------|
| GND | 通用地 |
| AGND | 模拟地（与 GND 同网络） |
| DGND | 数字地（与 GND 同网络） |

### 2.2 电源（Power Supply）
| 名称 | 电压 | 备注 |
|------|------|------|
| AVDD | 2.8V | 模拟电路供电（传感器像素阵列） |
| DVDD | 1.2V | 数字核心逻辑供电 |
| DOVDD | 1.8V | 数字 I/O 电平供电（也写作 IOVDD） |
| IOVDD | 1.8V | 同 DOVDD |
| AFVDD | 2.8V | 自动对焦音圈马达（VCM）供电 |
| VDDAF | 2.8V | 同 AFVDD |
| VCC | 任意 | 部分厂家对某路电源的简称 |

### 2.3 MIPI CSI-2 高速差分信号

> OV4689 支持 4-Lane MIPI，共 5 对差分线（4 路数据 + 1 路时钟）= 10 根信号线

| 名称（正极） | 名称（负极） | 含义 |
|-------------|-------------|------|
| MIPI_D0P | MIPI_D0N | 数据通道 0 差分对 |
| MIPI_D1P | MIPI_D1N | 数据通道 1 差分对 |
| MIPI_D2P | MIPI_D2N | 数据通道 2 差分对 |
| MIPI_D3P | MIPI_D3N | 数据通道 3 差分对 |
| MIPI_CLKP | MIPI_CLKN | 时钟通道差分对 |
| CSI_D0P | CSI_D0N | 同上，另一种写法（CSI 前缀） |
| CSI_D1P | CSI_D1N | 同上 |
| CSI_D2P | CSI_D2N | 同上 |
| CSI_D3P | CSI_D3N | 同上 |
| CSI_CLKP | CSI_CLKN | 同上 |
| D0P | D0N | 简写形式 |
| D1P | D1N | 简写形式 |
| D2P | D2N | 简写形式 |
| D3P | D3N | 简写形式 |
| CLKP | CLKN | 时钟简写形式 |

### 2.4 控制信号（Control）
| 名称 | 说明 |
|------|------|
| MCLK | 主时钟（Master Clock），主机输入给模组 |
| XCLK | 外部时钟（External Clock），同 MCLK |
| INCLK | 时钟输入（Input Clock），同 MCLK |
| CAM_MCLK | 带前缀的写法 |
| REFCLK | 参考时钟，同 MCLK |
| SCL | I²C 时钟线 |
| I2C_SCL | I²C 时钟线（带前缀） |
| CAM_SCL | I²C 时钟线（带前缀） |
| SDA | I²C 数据线 |
| I2C_SDA | I²C 数据线（带前缀） |
| CAM_SDA | I²C 数据线（带前缀） |
| RESET | 硬件复位，低电平有效（= 拉低复位） |
| RSTN | 同 RESET（N 表示低有效） |
| XSHUTDOWN | OmniVision 官方命名，低有效关断 |
| CAM_RST | 带前缀的复位 |
| NRST | 低有效复位 |
| PWDN | 掉电控制，高电平进入低功耗 |
| XSLEEP | 睡眠控制，同 PWDN |
| STDBY | 待机控制，同 PWDN |
| CAM_PWDN | 带前缀的掉电控制 |
| STBY | 待机，同 PWDN |
| FSIN | 帧同步输入，用于多摄像头时间同步 |
| FSYNC | 帧同步，同 FSIN |
| CAM_SYNC | 帧同步，带前缀写法 |
| STROBE | 闪光灯触发输出（模组 → 主机） |
| FLASH | 闪光灯触发，同 STROBE |
| CAM_STROBE | 闪光灯，带前缀写法 |

### 2.5 自动对焦（Auto Focus / VCM）
| 名称 | 说明 |
|------|------|
| AF+ | 音圈马达（VCM）正极 |
| AF- | 音圈马达（VCM）负极 |
| VCM+ | 同 AF+ |
| VCM- | 同 AF- |
| AFVCC | AF 电源正端（通常从排线引入 2.8V） |
| AFGND | AF 地 |

### 2.6 其他
| 名称 | 说明 |
|------|------|
| NC | No Connect，悬空，不接 |
| RESERVED | 预留，不接 |

## 四、对接开发板说明

### NanoPC-T4 / Firefly RK3399 （30-pin MIPI CSI-2 接口）

| MP4689-QZM01-V2 管脚 | 功能 | 对应 RK3399 30-pin 管脚 |
|---------------------|------|------------------------|
| GND | 地 | GND（多个） |
| AVDD | 2.8V | 开发板 2.8V 电源输出 |
| DVDD | 1.2V | 开发板 1.2V 电源输出 |
| DOVDD | 1.8V | 开发板 1.8V 电源输出 |
| SCL | I²C SCL | 对应 RK3399 CSI I²C SCL |
| SDA | I²C SDA | 对应 RK3399 CSI I²C SDA |
| MCLK | 24MHz | 对应 RK3399 MCLK 输出 |
| RESET | 复位 | 对应 RK3399 CAM_RESET |
| PWDN | 掉电 | 对应 RK3399 CAM_PWDN |
| MIPI_D0P/N | Lane 0 | 对应 RK3399 MIPI_CSI_D0P/N |
| MIPI_D1P/N | Lane 1 | 对应 RK3399 MIPI_CSI_D1P/N |
| MIPI_D2P/N | Lane 2 | 对应 RK3399 MIPI_CSI_D2P/N |
| MIPI_D3P/N | Lane 3 | 对应 RK3399 MIPI_CSI_D3P/N |
| MIPI_CLKP/N | 时钟 | 对应 RK3399 MIPI_CSI_CLKP/N |

> 需要一块"26-pin 转 30-pin"FPC 转接板，或者用转接座引出排针后飞线连接。
